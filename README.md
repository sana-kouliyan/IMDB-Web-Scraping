# IMDB Web Scraping & Dashboard
**Scraping IMDB movies, analyzing data, and building Excel dashboard**

## Overview
This project scrapes movie data from IMDB using Python, performs data cleaning and analysis, and creates a final dashboard in Excel to visualize trends such as top sales movies, genres, and etc.

## Project Structure
- `data/` → CSV output of scraped movies  
- `Scraping/` → Python scripts for crawl  
- `Dashboard/` → Excel dashboard  
- `README.md` → project description

## Skills Practiced
- Python Web Scraping (BeautifulSoup / Selenium)  
- Data cleaning and analysis (Excel)  
- Excel dashboards and visualization
